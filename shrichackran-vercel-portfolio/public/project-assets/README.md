# Project Assets

The asset modal reads project screenshots from these folders:

- `/project-assets/speech/` - Speech Denoising and Recognition System assets
- `/project-assets/lifelink/` - LifeLink Smart Blood Donation Platform assets
- `/project-assets/jobbzz/` - JoBzz AI Powered Job Matching System assets
- `/project-assets/wifi/` - currently empty because no WiFi Network Controller assets were provided

Paths are configured in `src/data/portfolioData.js`.
